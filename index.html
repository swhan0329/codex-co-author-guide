Repository-level Git hook pattern

Make Codex show up in your Git commits.

This repo documents how to use sample-codex-coauthor so commits automatically append Co-authored-by: codex <codex@openai.com>.

한국어 가이드 English guide Real example
What this is not
Not a Codex installer

You install Codex app or CLI once for yourself. You enable the co-author hook once per repository clone.

What changes
Only the commit trailer

You still do normal Git work yourself: add, commit, push, open a PR, and merge.

Two valid ways to use the original sample repo

Script path

Use the shell script directly

bash scripts/setup-codex-attribution.sh

Best default for beginners. Minimal, explicit, and easy to teach.

Skill path

Ask Codex to run the repo-local skills

  • $install-codex-coauthor-hook
  • $add-ai-coauthor
  • $show-ai-attribution-status

The original sample repo also supports a more agent-native flow.

Practical path

Keep only the minimum in production repos

  • .githooks/prepare-commit-msg
  • .gitmessage
  • scripts/setup-codex-attribution.sh

This is the approach used in the real repository example.
